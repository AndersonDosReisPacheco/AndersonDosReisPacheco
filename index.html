Sobre Mim

Jornada, paixão e compromisso com excelência

Transformando código em soluções impactantes

Como desenvolvedor Full Stack, combino expertise técnica em Java, Spring Boot e tecnologias web modernas para criar sistemas completos, desde APIs robustas até interfaces intuitivas. Minha abordagem integra back-end seguro e escalável com front-end responsivo, sempre focando em experiência do usuário, performance e boas práticas de desenvolvimento.

Tenho experiência prática no ciclo completo de desenvolvimento: concepção, arquitetura, implementação, testes e deploy em produção. Domino Java com Spring Boot para APIs REST, autenticação JWT, bancos de dados relacionais (PostgreSQL/MySQL), Docker básico, Git, e deploy em cloud (Render, Vercel, Netlify).

No front-end, desenvolvo com JavaScript moderno, HTML5 semântico, CSS3 avançado e React, criando interfaces responsivas, acessíveis e com ótima performance. Aplico Clean Code, arquitetura em camadas, SOLID e padrões RESTful para garantir qualidade, manutenibilidade e escalabilidade.
